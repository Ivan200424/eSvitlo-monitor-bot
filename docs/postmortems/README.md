# Postmortems

Ця директорія містить документи postmortem для Major та Critical інцидентів.

## Структура

Кожен postmortem зберігається як окремий markdown файл з назвою:
`YYYY-MM-DD-short-description.md`

## Як створити новий postmortem

1. Скопіюйте шаблон:
   ```bash
   cp ../POSTMORTEM_TEMPLATE.md ./YYYY-MM-DD-incident-name.md
   ```

2. Заповніть всі секції

3. Проведіть review з командою

4. Створіть action items як GitHub issues

## Архів

Всі postmortems зберігаються для історичного аналізу та навчання.
